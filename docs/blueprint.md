# **App Name**: Myanmar TikTok Lite

## Core Features:

- Video Upload: Users can upload short videos from their mobile devices.
- Video Feed: Display a feed of short videos, similar to TikTok's 'For You' page. Videos should autoplay, and the UI should keep track of whether videos are playing.
- Video Playback: Play back short videos smoothly. Users should be able to pause, play, and scrub through the videos.
- Basic Editing: Users can add music from the device library to the video.
- User Profiles: Create user profiles to allow video upload, liking and following.
- Text Translate: Uses generative AI tool to translate Myanmar video description text to other languages

## Style Guidelines:

- Primary color: Vibrant red (#E53935) to capture attention and reflect Burmese culture. 
- Background color: Light grey (#F5F5F5) to provide a clean and modern look.
- Accent color: Golden yellow (#FFC107) to highlight key elements.
- Font: 'PT Sans', a sans-serif font for clean and readable UI, suitable for both headline and body text.
- Use a set of modern, minimalist icons for navigation and key features.
- Simple, intuitive layout focused on video discovery and creation.
- Smooth transitions and subtle animations for a polished user experience.
